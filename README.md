# 🫀 Corpo Humano — A Ciência dos Órgãos Vitais

🔗 **[Acessar o Projeto](https://carlosguedes0007-maker.github.io/human-body/)**

## 📌 Sobre o Projeto
Uma experiência web interativa, educacional e imersiva dedicada a explorar a complexidade dos órgãos vitais do corpo humano. O site leva o usuário a uma jornada visual através da anatomia humana, revelando dados numéricos e curiosidades biológicas em uma interface moderna e profissional.

## 🚀 Tecnologias Utilizadas
- **HTML5** Semântico
- **CSS3** (Layout responsivo com Grid e Flexbox, animações CSS e variáveis para um design escalável)
- **JavaScript Vanilla** (Lógica de interatividade, controle de rolagem e manipulação do DOM)

## ✨ Principais Funcionalidades
- **Menu de Navegação Dinâmico**: Header inteligente com transparência em "glassmorphism".
- **Hero com Vídeo Interativo**: O background da tela inicial utiliza um vídeo otimizado (hero-video.mp4) que pode interagir de acordo com o scroll do usuário.
- **Cards Educativos Interativos**: Exibição detalhada de órgãos cruciais (Coração, Cérebro, Pulmões, Fígado, Rins e Estômago) com estatísticas do corpo humano renderizadas de forma legível.
- **Design UI/UX Responsivo**: Totalmente planejado para brilhar em qualquer tela, focando em paletas de cores imersivas, tipografia rica (fontes Inter e Outfit) e hierarquia visual focada na experiência do usuário.
